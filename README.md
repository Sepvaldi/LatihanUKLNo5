# LatihanUKLNo5

Program ini berfungsi untuk menghitung rata-rata nilai siswa berdasarkan jumlah siswa dan nilai yang dimasukkan oleh pengguna.

Pengguna diminta untuk memasukkan jumlah siswa yang nilainya akan dihitung lalu program akan menjalankan perulangan sebanyak jumlah siswa tersebut.Dalam setiap perulangan, pengguna diminta untuk memasukkan nilai siswa satu per satu, setelah nilai-nilai tersebut dijumlahkan ke dalam sebuah variabel, program akan menghitung rata-rata nilai siswa dengan cara mentotal nilai siswa dan dibagi dengan jumlah siswa.
